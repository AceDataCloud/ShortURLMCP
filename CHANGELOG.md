# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-08

### Added

- Initial release of MCP ShortURL server
- URL shortening with `shorturl_create` tool
- Batch URL shortening with `shorturl_batch_create` tool
- API information tool `shorturl_get_api_info`
- Usage guide tool `shorturl_get_usage_guide`
- MCP prompts for LLM guidance
- Bearer token authentication
- HTTP and stdio transport modes
- Docker support with health checks
- Kubernetes deployment configs
- Comprehensive test suite
- CI/CD with GitHub Actions
- PyPI publishing with CalVer
